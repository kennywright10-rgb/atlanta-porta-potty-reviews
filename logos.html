Usage Guidelines

  • Website Header: Use Full Horizontal Logo
  • Social Media: Use Square Logo (Facebook, Instagram, LinkedIn)
  • Browser Tab: Use Favicon (16x16 or 32x32)
  • Business Cards: Use Vertical Logo or Icon Only
  • Email Signature: Use Horizontal Logo (smaller size)
  • Print Materials: Use any logo, ensure 300 DPI for printing

File Formats

  • SVG: Best for websites (scalable, small file size)
  • PNG: For social media, email, documents (transparent background)
  • Recommended: Save both formats for flexibility

How to Use Logo on Your Website

1. Download the "Full Horizontal Logo" SVG
2. Save it as logo.svg in your project
3. Replace the text logo in your header with: <img src="logo.svg" alt="Atlanta Metro Porta Potty Reviews" height="60">
